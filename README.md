# masterpiece-game
Emulate the old 60s-70s game of buying/selling art.

# How to Run
1. Run `pip install -r requirements.txt`
1. Run `uvicorn app.main:app --reload`

READY!

Server runs at [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
